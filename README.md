## Controls

#### Player A
- **A / D** – Bevegelse venstre / høyre  
- **W** – Hopp  

#### Player B
- **← / →** – Bevegelse venstre / høyre  
- **↑** – Hopp  

### Stretch Tool
- **Q / E / M / N (+ Shift)** – Strekke / krympe terreng  
- **Venstreklikk** – Velg objekt  

---

## Beskrivelse av spillet

**Grow&DeGrow** er et co-op plattformspill der to spillere må samarbeide for å komme seg til døren og gå videre til neste level.

- **Player A (grønn)**
- **Player B (rosa)**

Ingen av spillerne klarer nivåene alene. For å komme videre må dere hjelpe hverandre ved å **strekke og krympe terreng og objekter** i banen.
Dette kan brukes til å lage plattformer, komme seg høyere opp eller åpne veier for den andre spilleren.
